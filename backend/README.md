# Plato Backend (FastAPI Starter)

A lightweight FastAPI starter for a Tinder-style restaurant recommender backend.

## What's included

- FastAPI app with versioned API router
- Environment-based settings via `pydantic-settings`
- CORS setup for local frontend dev
- Health check endpoint
- Example swipe and recommendation endpoints
- Basic tests with `pytest`

## Quickstart

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -e .[dev]
```

3. Copy environment file:

```bash
cp .env.example .env
```
